# Mathias's Gym Planner
### Feature 1: CSV Import
- Upload a Strong app workout export
- Parser validates and stores workouts in the database
- Preview before confirming

### Feature 2: Volume Tracking
- Queries your past 3 weeks of workouts
- Shows sets per muscle group in a horizontal bar table
- Also displays weight × reps as secondary metric

### Feature 3: Imbalance Flagging
- Automatically detects muscle groups with low volume
- Flags any group at <50% of median volume
- Highlights in red on the volume page

### Feature 4: Recommendations
- Pick a muscle group from a button grid
- Get 5 exercises ranked by how long since you did them
- Top recommendations are exercises you haven't done recently

### Feature 5: Exercise Database
- Browse all 50+ exercises by muscle group
- Filter by primary muscle group
- Foundational data for recommendations and volume tracking
